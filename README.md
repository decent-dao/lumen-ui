# Lumen UI

## Requires
`Chakra-ui` v2 or greater 
`React` v16, v17, v18

## install

```shell
$ npm install @decent-org/lumen-ui
```

## local development

### install dependencies

```shell
$ npm install
```

### run storybook
```shell
$ npm run storybook
```
